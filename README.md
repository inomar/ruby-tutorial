## Rubyの基礎
### ローカル変数
ローカル変数は小文字もしくは`_ruby`などアンダースコアから始まる

### グローバル変数
グローバル変数は`$`から始まる必要がある

### 定数
定数は大文字
定数はメッソドの中で定義できない

### インスタンス変数
`@message`のように@から始まる
インスタンスの中だけで参照できる変数

```
attr_accessor :インスタンス変数名
```

### self
メッソドの中では暗黙的にselfを参照している

### initilize
初期化
```
def initilize(length)
	@length = length
end
```

### クラスメソッド
`@@name`
クラスとそのインスタンスをスコープした変数をクラス変数
